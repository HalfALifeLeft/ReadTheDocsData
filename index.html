Commands

Welcome to the documentation for my bot, Data. Here you will find the documentation and necessary information to use all of his commands! This is a work in progress so it will be updated consistently, and it will look better eventually!

User Commands
Command Name Info
d!8ball <question> Ask the magic 8ball a question!
d!Avatar [@user or username] Get a user's avatar.
d!Balance [@user] Gets yours or a user's bank balance.
d!Booru <search term> Gets a random image from Safebooru.
d!Botinvite Get the link to invite Data to your own server!
d!Cuddle <@user or user> Cuddle a user
d!Daily NON-FUNCTIONAL - Collect your daily reward.
d!Gamble <$ to gamble> NON-FUNCTIONAL - Gamble your li- money away!
d!Help Get help with command for Data. Basically this but easier to code.
d!Hug <@user or user> Hug a user
d!Info The background info of Data, from Star Trek: The Next Generation
d!Invite Get an invite for the server, lasts 24 hours.
d!Issue Found a bug with Data? Report it to my github!
d!Kitty Get a random picture of a kitten.
d!Leaderboard Get the top 10 richest users in the server.
d!Meme Get a random meme from /r/memes.
d!Penis <@users or users> Get the user's penis lengths! 100% accurate!
d!Ping Get's the WebSocket's ping, averaged over 5 pings.
d!Serverstats Get the server's stats! How exciting! :o
d!Subreddit <subreddit name> Gets a random image from a subreddit of your choosing!
d!Suggestion Suggest a server change or whatever you want! That way you can feel like you are included!
d!Userinfo <@user or user> Get's a user's stats for the server. It was a pain to code, so enjoy it. Please.
Mod Commands
d!Ban <@user> Ban a user, requires the BAN_MEMBERS permission.
d!Kick <@user> Kick a user, requires the KICK_MEMBERS permission.
d!Mute <@user> Mute a user, requires the KICK_MEMBERS permission.
d!Purge 0 ≤ # ≤ 100 Purges the chat, requires the MANAGE_MESSAGES permission.
d!Unmute <@user> Unmutes a user if they are muted, requires the KICK_MEMBERS permission.
Admin Commands
d!Channel <channel name> Create a channel, requires the ADMINISTRATOR permission.
d!Config <config name> <config argument> Configure the server's configs, requires the ADMINISTRATOR permission.
d!Embed <hexcode> <embed content !!!MUST BE LESS THAN 1024 CHARACTERS!!!> Creates an embed, requires the ADMINISTRATOR permission.
d!Logs <Member Logging Channel> <Message Logging Channel> <Server Logging Channel> <Mod Logging Channel> Create and auto configure logging channels, requires the ADMINISTRATOR permission.
d!Roles See all the roles in the server, to grab their IDs, requires the ADMINISTRATOR permission.

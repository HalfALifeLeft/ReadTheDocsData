Commands

Welcome to the documentation for my bot, Data. Here you will find the documentation and necessary information to use all of his commands! This is a work in progress so it will be updated consistently, and it will look better eventually!

Mod Commands
Ban Ban a user, requires the BAN_MEMBERS permission.
Kick Kick a user, requires the KICK_MEMBERS permission.
Mute Mute a user, requires the KICK_MEMBERS permission.
Purge Purges the chat, requires the MANAGE_MESSAGES permission.
Unmute Unmutes a user if they are muted, requires the KICK_MEMBERS permission.
Admin Commands
Channel Create a channel, requires the ADMINISTRATOR permission.
Config Configure the server's configs, requires the ADMINISTRATOR permission.
Embed Creates an embed, requires the ADMINISTRATOR permission.
Logs Create and auto configure logging channels, requires the ADMINISTRATOR permission.
Roles See all the roles in the server, to grab their IDs, requires the ADMINISTRATOR permission.
